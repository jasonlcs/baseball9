# Retro Baseball - HTML5 Project (Final Version)

這是一個基於 HTML5 Canvas 的經典桌面機械棒球模擬遊戲，完全還原了 MLB 實體遊戲機的物理感，並整合了真實的棒球規則。

## Project Overview

*   **目標**：重現經典機械棒球遊戲的物理感、可愛的視覺回饋與完整的競技對決。
*   **技術棧**：HTML5, Vanilla JavaScript, CSS3。
*   **目前進度**：已完成所有開發階段（階段 1 ~ 5）。

## Core Features

1.  **物理與打擊系統**：
    *   **比例球棒**：具有握把、筒身與 3D 漸層質感的真實比例球棒。
    *   **節奏蓄力**：自動循環蓄力條（20%-100%），考驗玩家按下空白鍵的時機。
    *   **子彈時間**：進壘瞬間的慢動作補償，提升人類反應視窗。
    *   **雙重軌跡**：保留投球虛線與擊球實線，並標註落點。

2.  **真實棒球規則 (對齊標準規範)**：
    *   **SBO 系統**：精確計算好球、壞球與出局數。
    *   **進壘邏輯**：包含一安、二安、全壘打，以及「保送/觸身球」下的強迫進壘擠壘規則。
    *   **防守判定**：區分高飛接殺 (Fly Out) 與滾地刺殺 (Ground Out)。
    *   **九局賽制**：完整 1-9 局上下半場切換，並在終局宣佈勝負。

3.  **守備 AI 與角色輪替**：
    *   **一人分飾兩角**：上半局扮演投手（游標點擊瞄準），下半局扮演打者。
    *   **娃娃頭野手**：可愛的機械娃娃風格頭像，具備積極攔截、預判落點與步行歸位功能。

## Directory Structure

*   `index.html`: 遊戲主入口與仿古計分板 UI。
*   `style.css`: 復古數位風格樣式。
*   `game.js`: 核心遊戲邏輯、物理引擎、守備 AI 與規則計算。
*   `base.jpg`: 原始設計靈感參考圖。

## Usage

1.  在瀏覽器開啟 `index.html`。
2.  **角色操作**：
    *   **投手角色**：滑鼠點擊球場位置進行瞄準投球。
    *   **打者角色**：觀察蓄力條節奏，按下 `空白鍵` 或 `滑鼠左鍵` 揮棒。
3.  **防守完畢**：需等待野手全部走回原位後方可進行下一球。
